Ontario Government

The goal of this bootcamp project is to redesign the Ontario government website (Homepage and local page) for a more clear and simplistic way to access information by integrating new UI components.

Overview

Ontarians and new comers visit the Ontario.ca website to get information related to various government services. The Ontario Governement offers too many services and it can be challenging for users to find the information they need in short period of time. Our role was to redesign the website: homepage, local page and the navigation system to only have essential information and make it more user-friendly.


My Role: UX/UI Designer

Type: Group Project

Time: 2 Weeks

Problem

Overwhelming amounts of information make it difficult to find information and navigate around the website. The actual Ontario.ca website has too much content and writing which can not be easy to find information through categories.

Solution

Redesign the website for a more clear and simplistic way to access information by integrating new UI components will help users to find information quicly and easily. Adding illustations will create more spacing website to avoid having heavy text.

User Testing

Our objective is to know how users feel when navigating through the app and find the main features to perform the required tasks.

Tasks

  • Start a new shift, add a tip and end the shift.
  • View past, current and future job schedule in the calendar.
  • Chat with co-workers and manager.
  • Search for payment report and share the file.

Feedback

  • Use liked the edit option to update the tip amount.
  • User liked the chat feature to communicate internally.
  • User was able to perform all the tasks easily and successfully.
